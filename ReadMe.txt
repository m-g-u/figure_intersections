========================================================================
CONSOLE APPLICATION : figure_intersections Project Overview
========================================================================

This application reads a JSON file containing rectangle definitions and
outputs the list of the rectangles along with all their intersections.
The number of figures to process is limited by the system capacity.
This program uses a linear complexity and can be extended to other figure types
and rotations.
Supported C++ standards: C++03 and above.

Compilation: use MS Visual Studio with the main project file
or utilize stand-alone C++ compilers. 

Usage: figure_intersections <path to a JSON file>

Main files:

figure_intersections.vcproj
The main project file for VC++ projects. 

figure_intersections.cpp
The main application source file.

json.txt
A sample JSON file for testing.
